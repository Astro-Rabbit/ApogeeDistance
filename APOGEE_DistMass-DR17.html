Data Model: model

General Description
This Value-added catalog contains the absolute magnitudes, distances, and masses for dr17 stars predicted using a simple neural network. File also contains the stellar parameters used by the neural network. Select giant stars also have a predicted age calculated by comparing the mass to isochrones.
